# klasifikasi-gambar-sampah-organik-dan-anorganik
Implementasi visi komputer untuk pemilahan sampah otomatis: klasifikasi gambar sampah organik dan non-organik berbasis CNN (transfer learning MobileNetV2) serta web inferensi.
## Dataset
Dataset diambil dari Kaggle:
- Garbage Classification (12 classes)
  https://www.kaggle.com/datasets/mostafaabla/garbage-classification
